# MATH_MODEL_201709
 
2017年全国大学生建模竞赛B题 省二等奖

data文件夹里为题目数据

支撑材料文件夹里为程序与处理的数据图表

## “拍照赚钱”的任务定价
### 问题1：根据已结束的任务的数据，分析任务未完成的原因。
分析各影响任务定价的因素，采用多元回归模型得到初步定价规律，并用控制变量法进行检验，对得出的参数进行决策分析得到任务未完成原因。
### 问题2：提出一个新的任务定价方案，并与问题1原方案进行比较。
计算出当所讨论的变量发生改变时任务完成情况增加的概率，通过比较概率大小，设计新的任务定价方案。据此修正模型。
### 问题3：若实行多任务打包分布，修改定价模型，评价其影响。
引用聚类模型，对所有任务点按照相近距离和相近价格进行区域聚类，据此形成类包，对每个类包进行拟合，作为改变定价模型的依据。
### 问题4：针对新项目给出任务定价方案，并评价实施效果。
将新项目数据代入各模型，得到完成率。
